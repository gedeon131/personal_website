🛠️ Comment créer une API : Guide simple et pratique pour débutants

Créer une API (Application Programming Interface) est devenu une compétence essentielle pour tout développeur web. Une API permet à différentes applications de communiquer entre elles.

🔍 Qu’est-ce qu’une API ?

Une API est un ensemble de règles permettant à un programme d’accéder aux fonctionnalités ou aux données d’un autre. Par exemple :

  • Une app météo utilise une API pour récupérer la température.
  • Une app mobile utilise une API pour charger un profil utilisateur.

🧰 Technologies utilisées

  • Node.js
  • Express.js
  • Postman ou curl

🚀 Étapes pour créer une API simple

1. Installer Node.js

Téléchargez Node.js sur nodejs.org

2. Créer un projet

mkdir mon-api
cd mon-api
npm init -y

3. Installer Express

npm install express

4. Créer le fichier index.js

const express = require('express');
const app = express();
const port = 3000;

app.use(express.json());

app.get('/', (req, res) => {
  res.send('Bienvenue sur mon API !');
});

app.get('/users', (req, res) => {
  res.json([{ id: 1, nom: 'Alice' }, { id: 2, nom: 'Bob' }]);
});

app.post('/users', (req, res) => {
  const nouvelUtilisateur = req.body;
  res.status(201).json({ message: 'Utilisateur ajouté', utilisateur: nouvelUtilisateur });
});

app.listen(port, () => {
  console.log(`API en cours sur http://localhost:${port}`);
});

5. Lancer le serveur

node index.js

🧪 Tester l’API

Utilisez Postman pour tester les routes GET et POST.

🔁 Méthodes HTTP

Méthode Utilité
GET Lire des données
POST Créer une ressource
PUT Modifier une ressource
DELETE Supprimer une ressource

✅ Ce que vous avez appris

  • Les bases d’une API REST
  • Créer une API avec Express
  • Tester les routes HTTP

🚀 Pour aller plus loin

  • Connexion à une base de données
  • Authentification (JWT)
  • Organisation des fichiers

✍️ Conclusion

Avec Node.js et Express, créer une API devient simple. C’est une compétence fondamentale pour tout développeur moderne.
